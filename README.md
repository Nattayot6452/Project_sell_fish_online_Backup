# Docker commands
## Start docker
```
docker compose up -d
```

## Stop docker
```
docker compose down
```

# Git commands
## Download latest code
```
git pull
```

## Sent latest code to Github
```
git add .
git commit -m "xxxx"
git push
```

## In the case of first push
```
git push origin main
```

## See status
```
git status
```
